1. HTML Basic

1. HTML Document
    Một chương trình HTML luôn bắt đầu bằng: <!DOCTYPE html>, để khai báo loại tài liệu và phiên bản html mà trang web sử dụng.
    ** <!DOCTYPE> không phân biệt chữ hoa và thường
    Chương trình HTML bắt đầu bằng <html> và kết thúc bằng </html>.
    Nội dung của trang web sẽ được viết trong phần <body>

2. HTML Headings (Tiêu đề)
    Tiêu đề của 1 file HTML được định nghĩa bằng các thẻ từ <h1> đến <h6>.
    Thẻ <h1> chứa tiêu đề quan trọng nhất. Các thẻ về sau chứa tiêu đề ít quan trọng hơn.
 3. HTML Paragraphs (Văn bản trong HTML)
    Sử dụng thẻ <p>để có thể hiển thị văn bản trong 1 file html
 4. HTML Links(Đường dẫn trong HTML)
    Sử dụng thẻ <a> để hiển thị link trong html
    Đường link phải được đặt trong href.
